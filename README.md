# movie-recommendation-system
This project recommends movies to users based on content similarity using Machine Learning. Developed in Python with Google Colab, it uses data preprocessing, CountVectorizer, and Cosine Similarity to analyze movie descriptions and suggest similar movies, providing personalized and relevant recommendations.
# Movie Recommendation System

## Overview
This project recommends movies using Content-Based Filtering and Cosine Similarity.

## Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Algorithm
- CountVectorizer
- Cosine Similarity

## Features
- Movie recommendation
- Content-based filtering
- Similarity calculation
- Data preprocessing
